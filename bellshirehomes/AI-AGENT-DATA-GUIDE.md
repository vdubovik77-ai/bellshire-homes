# Bellshire AI-агент — інструкція з заповнення даних

Цей документ пояснює, як редагувати дані, якими користується голосовий/текстовий
асистент Олександра. Усе зібрано в **одному блоці** у файлі `index.html`.

## Де знаходяться дані

Відкрийте `bellshirehomes/index.html` і знайдіть (Cmd+F) маркер:

```
EDITABLE DATA
```

Під ним — 6 об'єктів. **Уже заповнені реальними даними** (нічого не треба робити):

| Об'єкт | Що це | Стан |
|--------|-------|------|
| `AI_MAP_ID` | id будинку → пін на карті | ✅ готово |
| `NEIGHBORHOOD` | школи / парки / магазини / транзит | ✅ заповнено |
| `MARKET` | ціна району, $/sqft, чому така ціна | ✅ заповнено |

**Потребують ручного заповнення** (нижче — як):

| Об'єкт | Що це | Стан |
|--------|-------|------|
| `PROPERTY_PHOTO_TAGS` | які фото = які кімнати | ⚠️ лише `exterior` |
| `VIDEO_CHAPTERS` | таймкоди фрагментів відео | ⚠️ порожньо |

> Якщо щось не заповнено — агент не ламається: відкриває повну галерею / грає
> відео з початку і чесно про це каже. Заповнення лише робить його точнішим.

## 1. Теги фото по кімнатах (`PROPERTY_PHOTO_TAGS`)

Дозволяє команді «покажи кухню / спальню / подвір'я» відкривати **саме те фото**.

### Як знайти номер фото

1. Відкрийте сайт → секцію **Our Homes** → клацніть на будинок (відкриється галерея).
2. Клацніть на потрібне фото — внизу лайтбокса видно лічильник, напр. **`7 / 50`**.
3. **Індекс = номер мінус 1.** Тобто фото `7 / 50` → індекс **`6`**.

### Як вписати

Знайдіть будинок за id і додайте індекси в потрібну категорію (можна кілька):

```js
'312':   {
  exterior: [0],          // фото 1
  kitchen:  [6, 7],        // фото 7 і 8
  master:   [12],          // фото 13
  bedroom:  [14, 15, 16],
  bathroom: [20],
  living:   [3, 4],
  dining:   [5],
  backyard: [45, 46],
  garage:   [49],
  view:     [2]
},
```

### Доступні категорії (як їх каже клієнт → ключ)

| Клієнт каже | Ключ |
|-------------|------|
| фасад, зовні, вид спереду | `exterior` |
| кухня | `kitchen` |
| головна / майстер-спальня | `master` |
| спальні | `bedroom` |
| ванна | `bathroom` |
| вітальня | `living` |
| їдальня | `dining` |
| подвір'я, задній двір | `backyard` |
| гараж | `garage` |
| вид, краєвид | `view` |
| цокольний / підвал | `basement` |
| офіс, кабінет | `office` |

> Достатньо вказати **по 1 фото** на категорію — агент відкриє його, а клієнт
> зможе гортати далі стрілками.

## 2. Таймкоди відео (`VIDEO_CHAPTERS`)

Дозволяє «покажи backyard на відео» — відео відкриється й **перемотається** на потрібний момент.

### Як знайти таймкод

1. Відкрийте відео будинку (кнопка ▶ на картці або через асистента).
2. Перемотайте до моменту (напр. подвір'я) і подивіться час на плеєрі: `0:45`.
3. Переведіть у **секунди**: `0:45` → `45`; `1:20` → `80`.

### Як вписати

```js
'312':  { backyard: 45, kitchen: 20, exterior: 5, living: 32, view: 58 },
```

Ключі ті самі, що й для фото (`backyard`, `kitchen`, `exterior`, `living`, `view`…).
